# Security

## Supported versions

| Version | Supported |
| ------- | --------- |
| 1.2.x   | Yes       |

## Reporting a vulnerability

Please **do not** open public GitHub issues for security problems.

Use **[GitHub private vulnerability reporting](https://github.com/pureframework/pure-framework/security/advisories/new)** and include:

- Description of the issue and potential impact
- Steps to reproduce (proof of concept if available)
- Affected versions

We aim to acknowledge reports within a few business days and will coordinate a fix and release before public disclosure when appropriate.

## Scope

This policy covers the `pureframework/pure-framework` library. Applications built with the framework are responsible for their own deployment, authentication, and server configuration — see [docs/security.md](docs/security.md).
